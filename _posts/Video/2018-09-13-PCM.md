# 前言

声音是模拟的连续信号，而计算机只能离散的存储。为了使得计算机具备音频的能力，必须支持连续音频信号的离散化描述，而PCM具备这个能力。



# PCM脉冲编码调制(Pulse Code Modulation）

脉冲编码调制就是把一个时间连续，取值**连续的模拟信号**变换成`时间离散，取值**离散的数字信号**后在信道中传输。`脉冲编码调制就是`对模拟信号先抽样，再对样值幅度量化，编码的过程。`

> 总结起来就是三个步骤：
>
> * 1、抽样；
> * 2、量化；
> * 3、编码；
